## 引

鄙人曾是一位 app 收集爱好者，收集了大量安卓安装包和项目地址，经朋友建议将其全部公开。大部分应用来自奇妙应用和酷安朋友的分享，感谢以上🙏。

我未来的目标是将该项目做成一个应用库app（但因为是基于github 所以仍需科学上网才能高速下载）
